# ylog Documentation

Welcome to **ylog** - Institutional Memory for Dev Teams.

ylog converts your GitHub PR history into discoverable context files throughout your codebase, powered by AI summarization.

## Quick Start

1. [Installation](/installation) - Install and set up ylog
2. [Authentication](/authentication/overview) - Configure GitHub access
3. [Configuration](/configuration/overview) - Customize ylog settings
4. Run `ylog sync` to generate context

## Core Concepts

- [Institutional Memory](/concepts/institutional-memory) - Understanding the problem ylog solves
- [Hybrid Output Strategy](/concepts/hybrid-output) - Database + context files approach
- [Context Files](/concepts/context-files) - Auto-generated .ylog files

## CLI Commands

- [`ylog sync`](/cli/sync) - Sync PR history and generate context files
